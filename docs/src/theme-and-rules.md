# Theme and Rules

The arena represents an agricultural cold-storage warehouse with pickup zones, drop zones, and docking zones.

Key constraints:

- Three holonomic robots operate in the arena.
- Crates are classified using ArUco marker IDs.
- Each crate must be placed in the correct drop zone.
- The run is limited to 250 seconds.
- The expected stack uses Ubuntu 22.04 and ROS 2 Humble.
